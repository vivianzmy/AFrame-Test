# A-Frame Test

Minimal static A-Frame scene served with a lightweight dev server.

## Run locally

- Ensure Python 3 is available: `python3 --version`
- Start the dev server:

```bash
npm run dev
```

- Open `http://localhost:5173` in your browser.

## Notes

- This uses Python's built-in HTTP server to avoid installing extra NPM dependencies.
- Edit `index.html` to modify the scene.
